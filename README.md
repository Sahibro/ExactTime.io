## ✨ Features
- **Atomic Precision:** Synced with global NTP servers for 0.001s accuracy.
- **Luxury UI:** Minimalist Deep Black & Gold aesthetic.
- **Global Coverage:** 50,000+ cities with automatic DST handling.
- **Developer API:** High-speed JSON time-sync for automation.
- **Zero Lag:** Built with Vanilla JS for ultimate performance.

## 🚀 How to use
Simply visit `exacttime.io` to see your local time vs. atomic time. Use the search bar to find exact times for any city worldwide.
